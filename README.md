
# Car Lane Game

Car Lane game implementation with HTML CSS HTMLCanvas and TS



## Run locally

1. clone the repo by

```
git clone https://github.com/BikramNischal/car-lane.git
```

2. change to project directory
```
cd car-lane
```

3. install dependencies
```
npm install 
```

4. build the project
```
npm run build
```

5. run the project
```
npm run dev
```
